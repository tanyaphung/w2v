# w2v

## Obtain titles from a list of snps and pubmed ids

```
python get_titles.py -h
usage: get_titles.py [-h] --snp_pubmed_cited SNP_PUBMED_CITED --outfile
                     OUTFILE

This script takes in a list of snps and pubmed id and return the title of the
pubmed article

optional arguments:
  -h, --help            show this help message and exit
  --snp_pubmed_cited SNP_PUBMED_CITED
                        REQUIRED.
  --outfile OUTFILE     REQUIRED.
```
